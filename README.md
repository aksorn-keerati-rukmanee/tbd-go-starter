## HOW TO START THE PROJECT

### "Run Hello"

```
go run ./cmd/hello
```

### Get Package

```
go get ./cmd/fiberserver
```

### Run

```
go run ./cmd/fiberserver
```

### Build

```
go build -o ./bin ./cmd/fiberserver
```
